# rkgk
A Pixiv browser.


## Compiling and running
Compiling requires either npm or yarn, and `make`.

```
$ yarn install
$ make
```

Running requires `config.ru` from [rkgk-elm](https://github.com/steinuil/rkgk-elm).

```
$ bundle install
$ rackup
```

Then open `index.html` with your browser.
